What I do...

Hello! My name is Abdullah Alghazali and I am currently 26 years old. I am pursuing a career in web development as I am very passionate about how websites work and all the new technology that comes along with it. I currently develop websites at a front end level with responsive and mobile web development as a top priority.

  • HTML
  • CSS
  • Javascript

HTML

I have been learning how to develop websites when I took my first course at Henry Ford College. HTML was a great way to jump start my developing career. It was very easy to learn and pretty self-explanatory. With the new iteration of HTML (5) developing websites was never easier! Learning all the new technologies that came along with HTML5 was an exciting challenge. My passion and joy for being able to develop websites at the highest level with your team is what excites me the most!

CSS

My CSS developr career goes hand-in-hand with my HTML career. CSS is ever so growing and so must we as developers. Being able to join your team will allow us to learn all the new ways we can change the web.

Javascript

I currently learning JavaScript development. Learning about arrays, functions, objects, and the DOM manipulation turned out to be very exciting! Learning Javascript has been a very fun ride! I did not realize how well Javascript goes along with HTML. While it also has been a rough ride, learning to master Javascript will go a long way in developing future web apps.
